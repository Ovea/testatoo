=== Item

*Item* is used for dropDowns and listBoxes.

[cols="3*", options="header"]
|===

|States|Properties|Intentions

|selected
|value
|select

|unselected
|
|unselect

|===
[source, java]
-------------------------------------------------------------------------------
DropDown dropdown = $('#dropdown') as DropDown
Item item = dropdown.item('Fedora')  // Or dropdown.items[1]
item.should {
    have value('Fedora')
    be unselected
}
-------------------------------------------------------------------------------
